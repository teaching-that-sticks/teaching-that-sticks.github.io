Working memory can only hold a few items at once.
When a slide, explanation, or task asks learners to track too many things simultaneously,
working memory fills up and learning stops — not because learners are disengaged,
but because the cognitive machinery that creates memories is at capacity.

## What this means for teaching

- **Reduce extraneous load** — remove anything on screen that isn't doing teaching work. Decorative images, dense text, and animations all cost working memory without buying learning.
- **Use both channels** — spoken and visual information travel through separate pathways. A diagram with a spoken explanation costs less than the same diagram with dense written labels.
- **Sequence complexity** — introduce one idea at a time. Let it land before adding the next.
- **Watch your own slides** — if you're reading from the slide, learners' eyes and ears are receiving the same information twice through different channels. That's not reinforcement; it's friction.
